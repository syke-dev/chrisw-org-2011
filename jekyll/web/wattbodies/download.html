Installation

Watt Bodies requires the August 2009 version (or later) DirectX SDK to be installed.
The SDK can be downloaded here: http://msdn.microsoft.com/en-us/directx/aa937788.aspx

Watt Bodies is playable on Windows XP / Vista / 7.
